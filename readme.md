## Key Features

* List Meals separated by day
* Create a Meal for specific day
* Track the diet planning
* Edit a Meal
* Delete a Meal
* View diet planning statistcs

## How To Use

```bash
# Install dependencies
$ npm install

# Run the app
$ npx expo start
```
